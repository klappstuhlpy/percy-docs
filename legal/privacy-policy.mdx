## Data We Collect

Percy only collects data that a feature actually needs to work. The table below lists every category, why it exists, and whether you can turn it off.

| Data | Purpose | Opt-out available? |
|---|---|---|
| **Discord user, server, channel, role, and message IDs** | Identify where your settings, content, and actions belong | No — required for the bot to function |
| **Presence history** (online / idle / dnd / offline transitions) | Presence graphs and statistics | Yes |
| **Username, nickname, and avatar history** | "Name history" and "Avatar history" features | Yes |
| **Server configuration and moderation logs** | Run the features a server admin enables (logging, automod, leveling, etc.) | Controlled by server admins |
| **Message and voice activity counts** | Leveling / XP system | Controlled by server admins |
| **Content you submit to a command** (reminders, tags, playlists, giveaways, notes, poll answers) | Provide that specific feature | When you stop using the feature |
| **Linked accounts** (e.g. AniList) **and timezone** | Features you explicitly opt into | Yes — remove the link at any time |

We **do not** sell your data. We **do not** use it for advertising or profiling. Every piece of data collected is used solely to make Percy's features work.

## How Long We Keep Your Data

- **Presence history** is automatically deleted after **30 days**.
- **Username, nickname, and avatar history** is kept until you delete it (see [Your Rights](#your-rights)) or until Percy is removed from the server.
- **Server configuration and moderation logs** are kept while Percy is in the server and deleted when Percy is removed.
- **Feature content** (reminders, tags, playlists, etc.) is kept until the item is completed, deleted by you, or no longer needed for the feature.

When data is no longer needed for its original purpose, it is deleted.

## Tracking: On by Default, Always Under Your Control

Presence, name, and avatar tracking are **enabled by default** so that related features work out of the box. You can change these settings at any time using Percy's `/settings` commands — no need to contact anyone.

| Command | What it does |
|---|---|
| `/settings show` | Opens your personal settings card with interactive toggle buttons |
| `/settings remove-personal-data` | Permanently deletes your stored presence, name, and avatar history |

Run `/settings show` and click the **Presence Tracking** or **Name & Avatar History** buttons to toggle each setting on or off instantly. A server administrator **cannot** consent to tracking on your behalf. These choices apply to your account across every server Percy is in.

## Third-Party Processors

Percy is hosted on secured private servers. Some features require sending limited data to external services. This only happens **when you use the relevant feature** — not by default.

| Service | What is sent | When |
|---|---|---|
| **Discord** | All bot activity (Discord is the platform) | Always |
| **Groq** | The text of a message you send to the AI assistant | When you use the AI assistant |
| **Google Translate** | The text you ask to translate | When you use translation |
| **AniList** | OAuth authorization to link your account | When you link AniList |
| **Lavalink** (self-hosted) | Music search queries and track URLs | When you use music features |
| **percy.klappstuhl.me dashboard** | Discord OAuth login and your dashboard configuration changes | When a server admin uses the web dashboard |

Each service processes data under its own privacy policy. Percy never sends a third party more data than the feature requires.

## Your Rights

You have the following rights over your personal data at any time. Many can be exercised directly through bot commands — no waiting, no forms.

<Note>
  These rights are aligned with the EU General Data Protection Regulation (GDPR). They cannot be waived, and nothing in this policy asks you to give them up. You also have the right to lodge a complaint with your local data protection authority.
</Note>

| Right | How to exercise it |
|---|---|
| **Access / export your data** | Run `/settings request-data` — Percy sends you a full JSON export |
| **Delete your data** | Run `/settings remove-personal-data` — permanently deletes your stored presence, name, and avatar history |
| **Opt out of tracking** | Run `/settings show` and click the **Presence Tracking** or **Name & Avatar History** toggle buttons |
| **Rectification** | Contact us at `klappstuhl65@pm.me` to request a correction to inaccurate data |

## Children

We do not knowingly collect data from anyone below that age. If we learn that we have, we will delete it promptly.
