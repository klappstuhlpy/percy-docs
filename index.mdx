Percy is an all-in-one Discord bot that brings powerful moderation, a fully-featured economy, high-quality music playback, server leveling, and a rich set of community tools to your server. Every command works as both a slash command (`/ban`) and a prefix command (`?ban`), so your team can use whichever style they prefer.

## What Percy Can Do

<Steps>
  <Step title="Invite Percy">
    Add Percy to your Discord server using the [invite link](https://discord.com/api/oauth2/authorize?client_id=1070054930125176923). Grant the recommended permissions so all features work out of the box.
  </Step>
  <Step title="Set up moderation">
    Use `/moderation` to enable AutoMod, configure the Sentinel verification system, set up an audit log channel, and define ignored channels or roles.
  </Step>
  <Step title="Enable the features you want">
    Turn on leveling with `/level config toggle`, start the economy with `/shop add`, or set up a music panel with `/music setup`. Each feature is independent — enable only what your server needs.
  </Step>
  <Step title="Customize to your community">
    Fine-tune command permissions with `/config`, change your server prefix with `/config prefix`, and let members control their own experience with `/settings`.
  </Step>
</Steps>
