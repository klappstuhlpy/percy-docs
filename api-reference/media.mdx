Manipulate and inspect images. Every endpoint accepts either a multipart `file` upload **or** a public image `url` that the server fetches on your behalf.

<Note>
  Private and reserved addresses are refused when you pass a `url` — the server only fetches public images.
</Note>

## Apply an effect

<ParamField path="POST /api/image/{op}" required>
  Scope: `images:read` · Content type: `multipart/form-data`
</ParamField>

<ParamField path="op" type="string" required>
  The effect to apply. One of: `blur`, `pixelate`, `deepfry`, `invert`, `grayscale`.
</ParamField>

**Query parameters**

<ParamField query="amount" type="number">
  Effect strength, interpreted per operation: `blur` → Gaussian sigma (default 8), `pixelate` → block size in pixels (default 16), `deepfry` → intensity 1–100 (default 50). Ignored by `invert` and `grayscale`.
</ParamField>

<ParamField query="share" type="boolean">
  When `true`, store the result and return a JSON `ShareResult` with a short shareable `/m/:id` link instead of the raw image bytes.
</ParamField>

**Body** — multipart form with a `file` upload or a `url` field.

```bash
curl -X POST "https://klappstuhl.me/api/image/blur?amount=12" \
  -H "Authorization: YOUR_API_KEY" \
  -F "url=https://example.com/photo.jpg" \
  --output blurred.png
```

Returns `200` with the processed image (`image/png`), or a `ShareResult` when `share=true`. Errors: `400` (bad input or unknown operation), `401`, `429`.

## Convert format

<ParamField path="POST /api/convert" required>
  Scope: `images:read` · Content type: `multipart/form-data`
</ParamField>

Transcode an image between raster formats.

**Query parameters**

<ParamField query="to" type="string" required>
  Target format. One of: `png`, `jpeg` (alias `jpg`), `webp`, `gif`, `bmp`, `tiff`.
</ParamField>

<ParamField query="quality" type="integer">
  JPEG quality, 1–100. Only used when `to=jpeg`. Defaults to 85.
</ParamField>

<ParamField query="share" type="boolean">
  When `true`, return a JSON `ShareResult` with a short `/m/:id` link instead of raw bytes.
</ParamField>

**Body** — multipart form with a `file` upload or a `url` field.

```bash
curl -X POST "https://klappstuhl.me/api/convert?to=webp" \
  -H "Authorization: YOUR_API_KEY" \
  -F "file=@photo.png" \
  --output photo.webp
```

Returns `200` with the converted image. Errors: `400` (bad input or unsupported target), `401`, `429`.

## Inspect metadata

<ParamField path="POST /api/metadata" required>
  Scope: `images:read` · Content type: `multipart/form-data`
</ParamField>

Read an image's dimensions, format, color type, and size without storing it.

**Body** — multipart form with a `file` upload or a `url` field.

Returns `200` with an `ImageInfo`:

```json
{
  "width": 1920,
  "height": 1080,
  "format": "png",
  "color": "Rgba8",
  "file_size": 482133
}
```

Errors: `400` (bad input or undecodable image), `401`, `429`.
