Check an uploaded file for malware. Percy runs the file through ClamAV and VirusTotal and returns a combined report.

## Scan a file

<ParamField path="POST /api/scan" required>
  Scope: `images:read` · Content type: `multipart/form-data`
</ParamField>

**Body** — multipart form with the file to scan, sent as a `file` field.

```bash
curl -X POST https://klappstuhl.me/api/scan \
  -H "Authorization: YOUR_API_KEY" \
  -F "file=@suspicious.bin"
```

Returns `200` with a `ScanReport` summarizing the ClamAV and VirusTotal results. Errors: `400` (no file provided), `401`, `429`.
