Host images on Klappstuhl.me and manage them programmatically.

## Upload images

<ParamField path="POST /api/images/upload" required>
  Scope: `images:write` · Content type: `multipart/form-data`
</ParamField>

Upload one or more image files.

**Query parameters**

<ParamField query="expires_in" type="integer">
  Optional time-to-live in seconds; the upload is auto-deleted afterwards (max 365 days).
</ParamField>

**Body** — a multipart form with the image file(s).

```bash
curl -X POST https://klappstuhl.me/api/images/upload \
  -H "Authorization: YOUR_API_KEY" \
  -F "file=@cat.png"
```

Returns `200` with an `UploadResult` describing the stored image(s). Errors: `400` (bad input), `401`, `403` (missing `images:write`), `429`.

## Delete an image

<ParamField path="DELETE /api/images/{id}" required>
  Scope: `images:write`
</ParamField>

<ParamField path="id" type="string" required>
  The image's ID.
</ParamField>

```bash
curl -X DELETE https://klappstuhl.me/api/images/abc123 \
  -H "Authorization: YOUR_API_KEY"
```

Returns `200` with a `DeleteResult`. Errors: `400` (invalid ID), `401`, `404` (not found), `429`.

## Bulk-download images

<ParamField path="POST /api/images/download" required>
  Scope: `images:read` · Content type: `application/json`
</ParamField>

Bundle several of your images into a single ZIP archive.

**Body**

```json
{
  "ids": ["abc123", "def456"]
}
```

An empty list bundles **all** of your images.

```bash
curl -X POST https://klappstuhl.me/api/images/download \
  -H "Authorization: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"ids": ["abc123", "def456"]}' \
  --output images.zip
```

Returns `200` with a ZIP archive (`application/zip`). Errors: `401`, `404` (no matching images), `429`.
