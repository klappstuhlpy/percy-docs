Admin-scoped endpoints expose homelab state. They require an `admin:read` or `admin:write` scope on your API key.

## List image-update status

<ParamField path="GET /api/admin/updates" required>
  Scope: `admin:read`
</ParamField>

Returns the per-service image update status for the homelab, sorted by service name.

```bash
curl https://klappstuhl.me/api/admin/updates \
  -H "Authorization: YOUR_ADMIN_API_KEY"
```

Returns `200` with an array of `ImageUpdate` objects. Errors: `401` (missing or invalid key), `403` (key lacks `admin:read`).
