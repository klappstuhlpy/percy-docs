## What's coming

A **public API** that lets you connect your own apps and services with Percy — think leaderboard widgets for your
community site, stream overlays, managing your server's tags from CI, or rewarding external events (donations,
subs, contributions) with in-server currency and XP.

When it ships, the interactive reference will be served at:

```
https://percy.klappstuhl.me/api/docs
```

## Integrate with Percy today

You don't have to wait to build on Percy's events. **Outbound webhooks** are available now: subscribe your
endpoint to server events (member joins, level-ups, moderation cases, ended giveaways, and more) and receive
HMAC-signed JSON deliveries with retries and a delivery health log.

<Card title="Outbound Webhooks" icon="webhook" href="/configuration/webhooks">
  Set up event subscriptions from the dashboard and verify signed deliveries.
</Card>

## For contributors

If you run Percy yourself and need the internal API reference, the bot serves an always-current interactive
(Scalar) reference at `http://127.0.0.1:8090/docs` whenever it runs with `INTERNAL_API_TOKEN` set.
